Projeto EcoScore

Este projeto, desenvolvido para a disciplina APS (Atividade Prática Supervisionada), tem como objetivo desenvolver um sistema interativo baseado em sustentabilidade por meio da linguagem de programação Python, a fim de incentivar os usuários a aderirem uma rotina saudável e benéfica ao planeta de forma divertida

Bibliotecas necessárias:
1- Datetime
2- Tkinter
3- Os
4- Customtkinter
5- Matplotlib
6- Pandas
